# 🖋️ Dev.tattoo

> **Sua história, contada através da arte.**

Site institucional do estúdio **Dev.tattoo**, focado em tatuagens personalizadas no estilo *graphic tattoo*. O projeto apresenta o trabalho do tatuador John Rod e convida clientes a transformarem suas histórias em arte na pele.

---

## 📋 Sobre o Projeto

O **Dev.tattoo** é uma landing page desenvolvida para divulgar os serviços de tatuagem personalizada do artista John Rod, com mais de 5 anos de experiência. O site tem como objetivo conectar clientes que buscam uma arte única e exclusiva, criada a partir de suas histórias pessoais, gostos e crenças.

---

## 🗂️ Estrutura do Site

O site é composto pelas seguintes seções:

- **Hero** — Apresentação principal com chamada para ação (*"Quero uma arte personalizada"*)
- **Proposta de Valor** — *"Suas Experiências Transformadas em Arte"*
- **Sobre o Artista** — Apresentação de John Rod e sua abordagem no estilo graphic tattoo
- **Galeria** — *"Histórias Eternizadas"* — portfólio com trabalhos realizados
- **Footer** — Créditos e direitos autorais

## 🚀 Como Rodar o Projeto

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/dev-tattoo.git

# Acesse a pasta
cd dev-tattoo

# Instale as dependências (se necessário)
npm install

# Rode o projeto localmente
npm run dev
```

Acesse em: `http://localhost:3000`
